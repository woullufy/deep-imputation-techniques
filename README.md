# Deep Imputation Techniques

This project provides simple deep learning methods for imputing missing data.

## Features
- Mean and k-NN imputation  
- Autoencoder imputation  
- Missingness generation  
- Basic plotting tools

## Notebooks
The notebooks contain the main results, including losses, reconstructions, and latent space plots.

## Requirements
- PyTorch  
- NumPy  
- Matplotlib
